# Convos Android

An end-to-end encrypted messaging app built on [XMTP](https://xmtp.org) with Jetpack Compose.

## Run the App

### Prerequisites
- Android Studio Ladybug or newer
- JDK 17
- Android SDK 29+

### Build & Run

1. Clone the repository
2. Open in Android Studio
3. Sync Gradle
4. Run the app (âŒ˜R or click Run)

Or via command line:

```bash
./gradlew assembleDebug
```

### Run Tests

```bash
./gradlew testDebugUnitTest
```
